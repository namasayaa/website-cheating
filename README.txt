# Pendeteksi Kecurangan Ujian by Adi Ageng Asy'ari

website ini dibuat dengan bahasa python dengan menggunakan framework flask yang berguna untuk membantu pengawas dalam melakukan pengawasan ketika ujian berlangsung. dengan bantuan machine learning unutuk menindetifikasi gerakan-gerakan yang mengarah kecurangan ketika ujian dengan bantuan ip cam.
contohnya gerakan :
menengok kanan dan kiri, menggunakan handphone, dan menunduk

## Installation

- Install Visual Studio Code dan buka folder website-skripsi. Lalu buka terminal pada Visual Studio Code, lalu menjalankan perintah berikut.
1. install Pyhton 3.11
2. install Flask sebagai framework
3. setelah itu install requirement.txt
4. kemudian install requirement_gpu.txt

- Install MySQL Workbench. Buat database baru dengan nama db_cheat dan import file db_cheat.sql 

- Mengatur koneksi database local dengan membuka _init_.py Atur DB_HOST, DB_PORT, DB_DATABASE, DB_USERNAME dan DB_PASSWORD sesuai dengan database local yang telah disiapkan.

- Kembali ke terminal, pada direktori proyek cheating-detection jalankan perintah "py main.py"


## Usage

Python V 3.11 up
Mysql Workbench 8.0.0
